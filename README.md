# Weather_App
# 🌤️ Compose Weather App

A beautiful, modern Android weather application built entirely with **Jetpack Compose**. This app provides real-time weather updates, daily forecasts, and hourly forecasts using a clean MVVM architecture.

## ✨ Features

- **Current Weather**: Get real-time temperature and weather conditions for your city.
- **Hourly Forecast**: Scroll through an hourly breakdown of the weather for the day.
- **Daily Forecast**: Plan ahead with a multi-day weather forecast.
- **Modern UI**: Features a sleek, dynamic dark-themed gradient background.
- **State Management**: Built using LiveData and `ViewModel` for robust lifecycle-aware data handling.
- **Dependency Injection**: Integrated with Dagger Hilt for scalable dependency injection.

## 🛠️ Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Toolkit**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Design System**: Material Design 3
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Injection**: [Dagger Hilt](https://dagger.dev/hilt/)
- **Asynchronous Programming**: Coroutines

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest version recommended)
- An active Weather API key (Currently configured for WeatherAPI)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/weatherapp.git
   ```
2. Open the project in **Android Studio**.
3. Add your API key in the `MainActivity` or build configuration.
4. Sync the project with Gradle files.
5. Run the app on an emulator or physical device.
